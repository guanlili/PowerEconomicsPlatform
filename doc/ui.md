# UI 设计规范

## 01. 标准色

### 主要颜色

| 颜色代码 | 使用场景 |
|---------|----------|
| #0066E9 | 主要视觉配色、点缀。用于主要文字、icon、导航栏标题颜色等 |
| #000409 | 用于重要级文字信息。内页正文文字、表格正文文字等 |
| #3B5F8D | 用于次要级文字信息。内页次要文字、分类名称等 |
| #D3D9E8 | 线框颜色。分割线、输入框线颜色等 |

### 次要颜色

- **常用辅助色**：表头文字颜色、小标题装饰色等

### 使用场景说明

- **主要视觉配色、点缀**：用于主要文字、icon、导航栏标题颜色等
- **重要级文字信息**：内页正文文字、表格正文文字等
- **次要级文字信息**：内页次要文字、分类名称等
- **线框颜色**：分割线、输入框线颜色等
- **分割线**：弹窗表格分割线颜色等
